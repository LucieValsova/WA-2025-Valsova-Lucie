Co jsou CSS jednotky?

CSS jednotky určují, jak budou vypadat rozměry a mezery v layoutu. Dělí se na absolutní a relativní.

Rozdíl mezi px a rem

Px je pevná jednotka, zatímco rem je relativní k velikosti písma kořenového prvku (`html`).

Jak funguje em?

Em se škáluje podle rodičovského prvku. Například tento text má padding `1em`, což odpovídá velikosti písma této sekce.

Použití vh a vw

Tento prvek má výšku `10vh`, což znamená, že se jeho výška mění podle velikosti okna.
